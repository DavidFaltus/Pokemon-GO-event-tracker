# Pravidla pro projekt Pokémon GO Event Tracker

Tento soubor obsahuje specifická pravidla a preference pro tento repozitář. Všichni vývojoví agenti by se jimi měli řídit.

## Funkční omezení
- **ŽÁDNÝ Plánovač, Poznámky a Checklisty u eventů:** Do detailů karet událostí (`EventCard.tsx`) neimplementujte ani nepřidávejte žádné textové poznámky, úkoly (tasks), checklisty ani plánovače. Uživatel tyto funkce výslovně nechce v aplikaci mít.
